처음에 Jekyll 테마 설치를 블로그들을 보면서 설치를 했었다.  
그런데 내 테마랑 맞지 않는 설치법을 보고 설치를 하다보니 GitHub에 Deploy오류가 난다던지 여러 오류로 오랜시간을 낭비하고 스트레스만 쌓였었다.
